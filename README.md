## 指南🧭  
1.以50题为一个package  
2.按照原始Leetcode题号索引  
3.争取一题多解，按照多种主题对题目进行分类  
4.主要以C++为主，后续考虑其他语言  
5.增加一些必要的知识点讲解  
6.重要程度标注✅  
7.难易程度标识✅  
8.欢迎程度标注✅  
9.题目出现时间，曾经出现过的轨迹图/表/时间线  
10.是否是公司题库  
11.涉及哪些知识点  
12.常规思路，进阶思路，优化思路，最受欢迎思路  
13.空间复杂度，时间复杂度，以及是否可以再次优化  
14.可以改进的空间；  
15.实际场景可能会有哪些应用；
